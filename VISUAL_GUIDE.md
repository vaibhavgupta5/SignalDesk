## Spacing System

Based on Tailwind's spacing scale:
- **xs**: 4px (0.25rem)
- **sm**: 8px (0.5rem)
- **md**: 12px (0.75rem)
- **lg**: 16px (1rem)
- **xl**: 24px (1.5rem)
- **2xl**: 32px (2rem)
- **3xl**: 48px (3rem)

## Typography

### Font Sizes
- **xs**: 12px - Hints, metadata
- **sm**: 14px - Body text, messages
- **base**: 16px - Default
- **lg**: 18px - Headings
- **xl**: 20px - Large headings
- **2xl**: 24px - Page titles

### Font Weights
- **normal**: 400 - Body text
- **medium**: 500 - Emphasis
- **semibold**: 600 - Headings
- **bold**: 700 - Strong emphasis
